Properties
Name Type Description
type string

The 'type' of the turn. This will be one of

  • move: a player has made a move
  • swap: a player has execute a tile swap
  • ended-game-over: the end of the game has been confirmed
  • challenge-failed: a challenge was not successful
  • challenge-won: a challenge succeeded
  • took-back: the last player took back their turn
player number

Index of the player who moved
nextToGo number

Index of the next player to play
deltaScore number | Array.<number>

If a number, change in score for player as a result of this turn. If an array, change in score for each player.
move Move

For move it indicates the Move. For took-back and challenge-won it is the Move just taken back/challenged.
challenger number

For 'took-back', 'challenge-won', index of the player who initiated the action
