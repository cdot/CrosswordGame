Module: game/findBestPlay

Calculate the best move in a Crossword game, given a dictionary, a game edition, a current board state, and a player tile rack. This should really be part of the 'Game' class, but is kept separate because we want to be able to run it in a worker thread. Loading it separately as a stand-alone function lets us both run it directly or in the worker by simply changing a dependency. The entry point to this module is the 'findBestPlay' function only.
