Class: Game

Game(params)

The Game object may be used server or browser side.

Constructor

new Game(params)

A new game is constructed from scratch by

new Game(...).create().then(game => game.onLoad(db)...

A game identified by key is loaded from a db by

db.get(key, Game.classes).then(game => game.onLoad(db)...

(may be null)

Parameters:
Name Type Description
params object

Parameter object. This can be another Game to copy game parameters, the result from Game.simple(), or a generic object with fields the same name as Game fields. Note that players and turns are not copied.
