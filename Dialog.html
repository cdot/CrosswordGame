Class: Dialog

Dialog(dlg, options)

Base class of modal dialogs. These are dialogs that support a set of fields and a submit button. In the HTML, any input or select that has a "name" attribute will be used to populate a structure that is posted to the server. The URL posted to is '/action' where 'action' is the result of a call to getAction on the Dialog.
