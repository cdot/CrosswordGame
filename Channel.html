Class: Channel

Channel()

Provides a simplex communications channel within a process.

Each Channel has a reference to a receiver, which must also be a Channel. By default, the receiver is this, which means calls to emit will invoke handlers added to this by calls to on.

this.receiver can be set to a different channel (call it B), in which case calls to emit will invoke handlers added to B by calls to B.on().

Two channels that are set to receive from eachother provide full duplex solution for end-to-end comms.
