Class: Player

Player(spec, factory)

A player in a Game. Player objects are specific to a single game, and are used on both browser and server sides.

new Player(spec, factory)

Parameters:

spec object

named parameters, or other Player object to copy. name and key are required. Any of _debug, isRobot, canChallenge, wantsAdvice,dictionary or missNextTurn can be passed. The player will be initialised with an empty rack (no squares).
