Class: Player

Player(params)

A player in a Game. Player objects are specific to a single game, and are used on both browser and server sides.

Player supports two different types of timed game; TIMER_TURN and TIMER_GAME. In a game configured for TIMER_TURN, each player has a turn timer, implemented in Player as a timeout that invokes a callback when the timer decays - see startTurnTimeout / stopTurnTimeout. For a TIMER_GAME, the player has a chess clock that runs while it is the player's turn, that is stopped when they are not making a play.
