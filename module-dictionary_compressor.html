Module: dictionary_compressor

Command-line program to generate a DAWG (Directed Acyclic Word Graph) from a word lexicon. Generates a somewhat optimised Trie, encodes it in an integer array, which it then gzips.

node js/dawg/dictionary_compressor.js will tell you how to use it.
