Methods

scorePlay(col, row, dcol, drow, tiles, wordsnullable) → {number}

Given a play at col, row, compute it's score. Used in findBestPlay, and must perform as well as possible. Read the description of analysePlay to understand the difference between these two related functions. Note: does not include any bonuses due to number of tiles played.

Parameters:
Name Type Attributes Description
col number

the col of the LAST letter
row number

the row of the LAST letter
dcol number

1 if the word being played across
drow number

1 if the word is being played down
tiles Array.<Tile>

a list of tiles that are being placed
words Array.<string> <nullable>

optional list to be populated with words that have been created by the play
