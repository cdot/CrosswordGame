Class: FileDatabase

FileDatabase(path, type)

Simple file database implementing Database for use server-side, where data is stored in files named the same as the key. There are some basic assumptions here:

  1. Callers will filter the set of keys based on their requirements
  2. Keys starting with . are forbidden
  3. Key names must be valid file names

Constructor

new FileDatabase(path, type)

Parameters:
Name Type Description
path string

name of a pre-existing directory to store games in, relative to requirejs.toUrl.
type string

will be used as the extension on file names
